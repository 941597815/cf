CF 鼠标宏 不限鼠标

#### 安装

安装 python 3.10

安装 tools 中的罗技驱动并禁用更新

安装 requirements.txt

管理员运行 run.bat

#### 游戏设置

分辨率 1600x900 全屏

HUD 放大 开

开火键改为 P 键

#### 功能

###### F5 开启/关闭 自动识别武器

###### F6 开启/关闭 自动准备挂机

###### F7 开启/关闭 红名开火

- Alt+B 开启/关闭 步枪模式

- Alt+L 开启/关闭 加特林模式

- Alt+J 开启/关闭 狙击枪模式

- Alt+U 开启/关闭 USP 模式

###### HOME 退出程序
